Minotar enables the use of Minecraft avatars in WordPress.

== Description ==

Minotar gives you the option to replace the default avatars found in WordPress with Minecraft avatars. The API used is found at http://minotar.net and can be used freely by anyone.

The avatars is taken from the Minecraft equivalent of users and comment posters, if a commenter uses the name Herobrine on a comment the skin of Herobrine in Minecraft is used as an avatar.

Note: The avatars are case-sensitive. Eg: notch and Notch are different names.


Features:

* Minecraft avatars for commenters and users
* BuddyPress support

== Installation ==

1. Upload the 'minotar-minecraft-avatars' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I get support? =

You can get support at #axxim on irc.esper.net

= How do I change the avatars back? =

Either you change the default avatar in Settings - Discussion on the admin panel or you just delete the plugin.
